## 1. Getting Started

To get started, clone the repository and set up the virtual environment:

```sh
$ git clone https://github.com/k-onoue/c-bbo.git
$ cd c-bbo
$ python3 -m venv .venv
$ source .venv/bin/activate
$ pip install --upgrade pip
$ pip install -r requirements.txt
```
